# Grupo FRK

Site institucional do Grupo FRK, empresa focada em soluções digitais para pequenas empresas.

## Link do projeto online

Acesse o site publicado:

https://grupofrk.github.io/grupo-frk-site/

## Objetivo do projeto

Apresentar os serviços do Grupo FRK, incluindo criação de sites, cardápios digitais e automações com IA.

## Serviços apresentados

- Sites profissionais
- Cardápios digitais
- Automações com IA

## Portfólio

O site apresenta o projeto Porto Grill como primeiro item de portfólio do Grupo FRK.
